  • absolute: 절대 위치 (부모 요소 중에 position 속성이 static이 아닌 요소를 기준으로 위치)
  • relative: 상대 위치 (자기 자신을 기준으로 위치)
  • fixed: 고정 위치 (브라우저를 기준으로 위치)
  • static: 기본 위치 (요소를 일반적인 문서 흐름에 따라 배치)
  • sticky: 스크롤 영역을 기준으로 위치 (스크롤 영역을 벗어나면 fixed와 같은 효과)
  • z-index: 요소의 쌓임 순서를 지정 (숫자가 클수록 위에 쌓임)
  • z-index: 0; // 기본값
  • z-index: -1; // 가장 아래에 쌓임
  • z-index: 1; // 가장 위에 쌓임
  • z-index: 999; // 가장 위에 쌓임
